#Leet Code in C
----
##Coding Style
Use standard BSD style for the source code.

```
bsdstyle()
{
    astyle --style=bsd --indent=spaces=4 $*
}
```

##Build Code
```
$ make clean
$ make all
```

##Execute Code
```
$ rel/leet
```
